Chicago Public Library Cardと無料・割引特典

最終更新：2026年4月25日

Chicago Public Library Cardとは

シカゴ市内に住んでいる場合、Chicago Public Library（CPL、シカゴ公共図書館）の図書カードを無料で作ることができます。図書カードがあると、本やDVDを借りられるだけでなく、電子書籍、オーディオブック、オンライン学習、新聞・雑誌、ミュージアムの無料・割引パスなどを利用できます。

  • Chicago Public Library
  • Get a Library Card

図書カードを作るには、CPLの各locationで申請します。名前・写真・Chicago住所が確認できる有効なIDがあると手続きがスムーズです。Illinois driver’s license、State IDなどが代表例です。Chicago住所入りの写真付きIDがまだない場合は、写真付きIDに加えて、名前とChicago住所が記載された郵便物などを持参します。CPL公式FAQでは、郵便物、utility bill、personal check、voter registration card、student ID、passportなどが例示されています。

  • Library Cards FAQ

14歳未満の子どもも図書カードを作ることができます。その場合は、親またはguarantorの署名と、住所が確認できるIDが必要です。14〜17歳のteensは大人と近い権限のカードを持てますが、18歳未満はDigital Museum Passesを予約できません。

Water Works Branch

Northwestern University Chicago campus周辺から近いCPLの一つは、Water Works Branchです。

Water Works Branch

163 E. Pearson Street

  • Chicago, IL 60611

Water Works Branchは、Northwestern Chicago campusやStreeterville周辺から近く、図書カード作成や本の利用には便利です。ただし、2026年4月25日時点のCPL公式ページでは、Water Works BranchはWi-Fi、public computers、printingを提供していないと案内されています。パソコン利用や印刷が目的の場合は、Harold Washington Library Centerなど別のlocationを確認してください。

Harold Washington Library Center

400 S. State Street

  • Chicago, IL 60605

Full Access Library CardとTemporary Access Library Card

2025年5月1日以降、CPLではカード制度が一部変更されています。CPLはeCardの新規発行を終了し、通常のFull Access Library Cardと、書類がまだ十分にそろっていない人向けのTemporary Access Library Cardを案内しています。Temporary Access Library Cardは90日間有効ですが、更新はできず、OverDrive / Libby、hoopla、Wi-Fi hotspot、museum passesなどは利用できません。

  • Library Card Changes

Full Access Library Cardでは、physical items最大50点、physical holds最大12件、OverDrive / Libby最大10点、OverDrive holds最大5件、hoopla月5点、Digital Museum Passes、Wi-Fi hotspotなどが利用対象に含まれます。Full Access Library Cardは、Chicago在住である限り有効期限がなくなったと案内されています。

Digital Museum Passes

Chicago Public Library Cardの大きな特典の一つが、Digital Museum Passesです。これは、CPLカード番号とPINでログインし、ミュージアム、動物園、歴史施設などの無料・割引パスをオンラインで予約できるサービスです。

  • Digital Museum Passes
  • Digital Museum Passes FAQ

予約はオンラインで行います。CPL公式FAQでは、library card numberとPINでログインし、日付、施設名、場所などから利用可能なofferを検索できると案内されています。CPLのPINは、初期設定ではZIP codeです。

Digital Museum Passesを予約できるのは、18歳以上でphysical CPL library cardを持っている人です。予約した本人が当日同行し、予約名と一致するphoto IDを求められる場合があります。

Digital passは予約した日付にのみ有効です。予約は最大3か月先、具体的には現在の月と次の2か月分まで可能です。また、同時に持てるactive reservationは3件までです。

施設によって、スマートフォン画面の提示でよい場合と、印刷したpassが必要な場合があります。確認メールに、mobile passでよいか、printed passが必要かが記載されます。印刷またはダウンロード後は変更しにくくなるため、実際に行ける日が確定してからprint / downloadするのが安全です。

Kids Museum Passport

CPL公式サイトには、Digital Museum Passesとは別にKids Museum Passportも掲載されています。Kids Museum Passportは、Chicago在住の大人がcheck outできる制度で、大人2名までと18歳未満の子どもを含むグループが、Chicagoの文化施設を無料で利用できると説明されています。対象は15施設と案内されています。

  • Kids Museum Passport / Programs and Partnerships

ミュージアム関連の制度は、Digital Museum Passes、Kids Museum Passport、施設ごとのofferが並行して案内されているため、実際に利用する際は、CPL公式ページとDigital Museum Passesの予約画面で、対象施設、人数、子どもの同伴条件、印刷の要否を確認してください。

予約時に確認すること

Museum passを使う場合は、施設ごとに条件が異なります。予約前に、無料なのか割引なのか、対象人数、大人の人数上限、子どもの同伴が必要か、特別展が含まれるか、駐車場代が含まれるか、スマートフォン画面でよいか、印刷が必要か、予約者本人のphoto IDが必要かを確認してください。

人気施設は枠が早く埋まることがあります。日程が決まったら、早めにDigital Museum Passesを確認するとよいです。

電子書籍・オーディオブック・雑誌

CPLカードがあると、Libby / OverDriveで電子書籍、オーディオブック、雑誌を借りられます。英語学習中の方には、子ども向けの本、音声付きの本、簡単な英語の本を探すのにも便利です。

  • OverDrive / Libby
  • CPL Online Resources

CPLのonline resourcesには、Mango Languages、Morningstar、LinkedIn Learning、New York Times、Chicago Tribune archives、eBooks、magazinesなどが含まれています。

hoopla

hooplaでは、映画、音楽、オーディオブック、電子書籍などを利用できます。Full Access Library Cardでは、2025年5月以降、hooplaは月5点までと案内されています。

  • hoopla

New York Times、LinkedIn Learning、Mango Languagesなど

CPLのonline resourcesでは、New York Times、LinkedIn Learning、Mango Languagesなども利用できます。New York Timesはニュースや英語学習に、LinkedIn Learningはソフトウェア・ビジネス・デザイン・プログラミングなどの学習に、Mango Languagesは語学学習に使えます。

  • CPL A-Z Resources
  • CPL Online Resources

Mango Languagesは、大人だけでなくkids、teens、tweens向けの語学学習リソースとしても掲載されています。

パソコン、印刷、スキャン

CPLでは、多くのlocationでpublic computer、printing、scanningを利用できます。図書カードがない場合でも、daily guest passでcomputer accessを使える場合があります。CPLのcomputerには、Microsoft Office、Adobe Reader、Gimp、Paint.net、OpenShot、Notepad++などが入っていると案内されています。

  • Use a Computer or Printer

印刷は白黒のみで、2026年4月25日時点では、片面15セント、両面25セント、1日あたり最初の$1.50分は無料です。スキャンは無料で、USB保存、メール送信、Google Driveへのアップロード、QR code経由でのスマートフォンへのダウンロードなどが可能です。

ただし、Water Works Branchのように、Wi-Fi、public computers、printingを提供していないlocationもあります。印刷やパソコン利用が目的の場合は、事前に各locationページで設備を確認してください。

Wi-Fi hotspot / Internet to Go

CPLには、Internet to Go WiFi Lendingというプログラムがあります。CPL cardholderでChicago residentの14歳以上であれば、対象locationでWi-Fi hotspotを借りられる場合があります。貸出期間は3週間で、誰もholdしていなければ最大15回まで自動更新される場合があります。

  • Internet to Go WiFi Lending
  • Borrow a WiFi Hotspot

Wi-Fi hotspotは全locationで借りられるわけではありません。CPL公式FAQには、Austin、Brighton Park、Canaryville、Chicago Lawn、Legler Regional、North Pulaski、Woodson Regionalなど、hotspotを扱うlocationが掲載されています。pickup locationもhotspotを扱うbranchにする必要があります。

CPS児童・生徒向け：The 81 Club

Chicago Public Schools（CPS）の児童・生徒には、The 81 Clubというstudent cardがあります。これは、通常のCPL cardよりも作りやすい学生向けカードで、学生がIDや親の署名を用意するハードルを下げるための制度です。

  • The 81 Club
  • The 81 Club FAQ

ただし、81 Club student cardは通常のFull Access Library Cardより制限があります。CPL FAQでは、81 Club student cardではphysical itemsは5点までで、museum passes、hotspots、Chromebooksは利用できないと説明されています。ミュージアムパスを予約する場合は、基本的に18歳以上の大人のFull Access Library Cardを使います。

CityKeyという選択肢

Chicago Public Library Cardとは別に、Chicago CityKeyという市発行のIDカードもあります。CityKeyはChicago residents向けの無料の任意IDで、government-issued ID、Chicago Public Library Card、CTA Ventra Card、Chicago Rx prescription drug discount cardとして使えると案内されています。移民ステータス、住宅状況、年齢、性別にかかわらず、Chicago residentsが申請できる制度です。

  • Chicago CityKey
  • Applying for your CityKey
  • CityKey FAQ

CityKeyはLibrary Cardとしても使えるため、Illinois driver’s licenseやState IDをまだ持っていない方にとって、Chicagoの公的IDとして選択肢になる場合があります。ただし、CityKeyはdriver’s licenseの代わりにはならず、飛行機搭乗用IDとしても使えません。申請にはproof of identityとproof of residencyが必要です。

使い方の例

家族でChicagoに来た直後は、まずCPLカードを作っておくと便利です。Museum passを使えば、ミュージアム、動物園、歴史施設などに無料または割引で行ける可能性があります。対象施設や条件は時期によって変わるため、Digital Museum Passesでその都度確認してください。

英語学習にはLibby / OverDriveの子ども向け電子書籍、Mango Languages、LinkedIn Learning、New York Timesなどが使えます。印刷やスキャンが必要なときも、CPLのlocationを活用できます。CPSに通うお子さんにはThe 81 Clubがありますが、museum passなどには制限があるため、家族で施設を利用する場合は大人のFull Access Library Cardを作っておくと便利です。
