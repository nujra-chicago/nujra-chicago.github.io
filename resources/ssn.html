Social Security Number (SSN)について

最終更新：2026年4月24日

Social Security Number（SSN）は、アメリカで給与、税務、社会保障、銀行口座、クレジットカード、住居契約、携帯電話契約などに幅広く使用される9桁の個人識別番号です。日本のマイナンバーに近い役割を持ち、渡米後の生活セットアップを進める上で非常に重要です。

Northwestern Universityで給与を受け取る研究者、ポスドク、職員などは、原則としてSSNの申請が必要です。SSNがあると、大学での給与・税務手続きに加えて、銀行口座、クレジットカード、運転免許証・州ID、アパート契約、携帯電話契約などの手続きが進めやすくなります。

渡米直後は各種手続きが重なるため、申請可能になったら早めにSSNを取得しておくことをおすすめします。

雇用・給与・税務関連

Northwesternで給与を受け取る場合、SSNはPayroll、税務書類、雇用記録の管理に使用されます。SSN取得後は、NorthwesternのHR/Payroll情報を更新します。

銀行口座・クレジットカード

銀行口座の開設、クレジットカード申請、ローン申請などでSSNを求められることがあります。SSNがあると金融機関での本人確認がスムーズになり、アメリカでのクレジットヒストリー構築にも役立ちます。

運転免許証・州ID

運転免許証や州IDの申請でも、SSNがあると手続きが進めやすくなります。車を運転する予定がある方、身分証として州IDを取得したい方は、早めにSSNを取得しておくと便利です。

住居・携帯電話・各種契約

アパート契約、携帯電話、インターネット、電気・ガスなどの契約時にもSSNを求められることがあります。SSNがあると本人確認や信用確認が進めやすくなり、保証金や追加書類の面で有利になる場合があります。

SSN申請のタイミング

Northwesternの案内では、J-1 scholar、J-1 student intern、H-1B employeeなどは、米国入国後10日経過してからSSN申請が可能とされています。

J-1 scholarの場合は、DS-2019上のprogram start date以降、かつOISS check-in完了後に申請します。

J-1 Short-Term Scholar、J-1 Research Scholar、J-1 Professorの場合、Northwestern OISSのSSN authorization letterは不要と案内されています。F-1/J-1 studentやJ-1 student internでは追加書類が必要になる場合があります。

SSN申請手順

必要書類を準備する

主な必要書類は以下です。

  • パスポート
  • ビザ
  • DS-2019
  • I-94
  • NorthwesternのJob Offer Letterまたは雇用証明書
  • SSAオンライン申請後のconfirmation number（O number）

I-94はこちらから取得できます。
https://i94.cbp.dhs.gov/

SSA公式サイトからオンライン申請を開始する

SSA公式サイトの「Request Social Security number for the first time」ページから申請を開始します。

SSA公式ページ：
https://www.ssa.gov/number-card/request-number-first-time

上記ページ内の「Get started」からオンライン申請へ進みます。

直接オンライン申請ページへ進む場合はこちらです。
https://secure.ssa.gov/ossnap/public/landingOSsnap

オンライン申請後に表示されるconfirmation number（O number）を保存または印刷してください。Northwestern OISSは、オンライン申請後45日以内にO numberと必要書類を持ってSSA officeへ行くよう案内しています。

SSA officeへ行く

O numberと必要書類の原本を持って、Social Security Administration（SSA）officeへ行きます。

最寄りのSSA officeはこちらから検索できます。
https://www.ssa.gov/locator/

Northwesternの案内では、Evanston campus近くでは2116 Green Bay Road、Chicago campus近くでは605 W Washington BlvdのSSA officeが紹介されています。訪問前に、上記のSSA office locatorで住所・受付状況・営業時間を確認してください。

SSN cardの到着を待つ

申請後、SSN cardは申請時に記載した住所へ郵送されます。通常の処理期間は2〜4週間程度と案内されていますが、追加確認が必要な場合はさらに時間がかかることがあります。

注意点

  • SSN申請は無料です。
  • 氏名はパスポート、DS-2019、I-94などの表記と一致するように記入してください。
  • SSN cardは重要な個人情報なので、普段は持ち歩かず、安全な場所に保管してください。
  • SSN取得後は、NorthwesternのHR/Payroll情報を更新してください。
  • OISSはSSN cardのコピー提出を求めないと案内しています。SSNをメール本文に書いたり、カード画像をメール添付で送ったりしないよう注意してください。
